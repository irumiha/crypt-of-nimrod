# Assets

This is the **canonical** copy of the game's art and audio. Every chapter
directory under `code/` carries its own identical copy (so chapters are
standalone); after changing anything here, run `tools/sync_assets.sh` to
propagate it. CI fails if the copies drift.

All assets are CC0 (public domain).
